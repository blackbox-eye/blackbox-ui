# Sprint planning – Opgavestyring (AIG)

**Planlæg næste sprint/opgavepakke i ALPHA Interface GUI (Blackbox E.Y.E).**

• Start med sprint backlog-tabel: Opgave, ansvarlig, estimeret tid, afhængigheder, prioritet.
• Del opgaver efter arkitektur, integration, test/QA, compliance, dokumentation, deployment.
• Tag udgangspunkt i handlingsplan, AIG Masterprompt og seneste statusrapport.
• Inkludér altid sikkerheds-, logging-, compliance- og audit-opgaver som egne punkter.
• Svar i matrix/punktform og med PDF/README-henvisning til beslutningsgrundlag.
• Versionér altid rapporten (vX.Y, dato, ansvarlig).

---
**Versionering:**  
Denne rapport er versioneret efter AIG-standard:  
`/docs/reports/v{X.Y}_YYYYMMDD_statusrapport.md`  
Husk at opdatere changelog, ansvarlig og versionsnummer for audit/trace.
