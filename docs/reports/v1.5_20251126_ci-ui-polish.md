## 🔧 Technical Details

### Workflow Improvements Applied

1. **Concurrency Control**
   ```yaml
   concurrency:
     group: ci-deploy-${{ github.ref }}
     cancel-in-progress: false
   ```

2. **Timeout Protection**
   - Build & Verify: 20 min
   - Delete index.html: 15 min
   - FTP Deploy: 20 min
   - Smoke Tests: 25 min

3. **Secrets Validation**
   - Validates FTP_HOST, FTP_USERNAME, FTP_PASSWORD, FTP_REMOTE_PATH
   - Validates CF_ZONE_ID, CF_API_TOKEN (optional)
   - Fails fast if required secrets missing

4. **Path Filters**
   ```yaml
   paths-ignore:
     - '.github/agents/**'
     - 'docs/**'
     - '**/*.md'
   ```

### CSS Design System Compliance

Blackbox EYE Assistant widget nu bruger:
- `--form-field-bg` for input background
- `--form-field-border` for borders
- `--form-field-text` for text color
- `--form-field-placeholder` for placeholder
- `--primary-accent` for focus states
- `--primary-accent-soft` for focus shadow
