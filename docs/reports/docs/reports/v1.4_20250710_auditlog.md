# Projekt-startprompt – ALPHA Interface GUI (AIG)

Kopiér og indsæt dette hver gang du starter en ny AIG-session i ChatGPT eller i projektstyringsflow:

---
Du skal nu løse en udviklings-, arkitektur-, sikkerheds- eller compliance-opgave for ALPHA Interface GUI (Blackbox E.Y.E).

Tag udgangspunkt i de centrale projektfiler:
• README.md – AIG
• SYSTEM_BLUEPRINT.md/PDF – AIG
• AIG Masterprompt v1.0 (denne fil)

**Svarets format og krav:**
1. Start med matrix, punktform eller tabel over flows, moduler, features og sikkerhedskrav.
2. Angiv altid kilde (README, PDF, sektion) for hvert hovedpunkt.
3. Udarbejd altid manual/workflow fra setup til test/deploy (inkl. CI/CD, sikkerhed, QA).
4. Output skal være audit-egnet og klar til revision eller onboarding – ingen løse forklaringer.
5. Brug kun de nyeste projektfiler i AIG-projektet.
6. Ved fortsættelse: Opsummer sidste status, hent relevante outputs, fortsæt fra sidste trin, og brug samme format.

---

`/docs/reports/v{X.Y}_YYYYMMDD_statusrapport.md`
