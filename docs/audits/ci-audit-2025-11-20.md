## 4. Hardening Recommendations

### 4.1 HIGH PRIORITY: Fix index.html Deletion

**Problem**: File not actually being deleted despite command success

**Recommended Fix**:
```yaml
- name: Delete index.html via FTPS
  run: |
    lftp -c "
      set ftp:ssl-force true;
      set ftp:ssl-protect-data true;
      set ssl:verify-certificate no;
      set ftp:passive-mode on;
      
      open -u $FTP_USERNAME,$FTP_PASSWORD $FTP_HOST;
      cd $FTP_REMOTE_PATH || exit 1;
      
      # Delete with explicit verification
      rm -f index.html;
      sleep 2;  # Wait for server to process
      
      # Fail if still exists
      ls -la index.html 2>/dev/null && exit 1 || echo 'Confirmed deleted';
      
      bye
    "
```

### 4.2 HIGH PRIORITY: Pin Actions to SHA

**Recommended Changes**:
